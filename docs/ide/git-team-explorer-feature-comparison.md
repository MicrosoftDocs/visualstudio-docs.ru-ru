# <a name="side-by-side-comparison-of-git-and-team-explorer"></a>Параллельное сравнение Git и Team Explorer

Мы запустили первую версию нового интерфейса Git в Visual Studio 2019 [версии 16.8](/visualstudio/releases/2019/release-notes/). Этот новый интерфейс позволяет сократить переключение контекста с помощью простого окна **Изменения Git**, включающего общие задачи Git. Он также содержит окно **Репозиторий Git** на уровне экрана для выполнения более сложных операций Git, таких как управление филиалами и просмотр репозитория.

Если вы используете Team Explorer, ниже приведено пошаговое руководство, в котором объясняется, как можно использовать новый интерфейс Git.

> [!NOTE]
> В следующем разделе приведены снимки экрана, размеры которых соответствуют ширине столбцов таблицы. Щелкните каждый снимок экрана, чтобы просмотреть его более крупную версию. (Если вы используете устройство с сенсорным экраном, то для просмотра более крупной версии снимка коснитесь его.)
