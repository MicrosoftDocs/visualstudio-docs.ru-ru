# <a name="ca0055"></a>CA0055

Не удалось загрузить файл

Средству анализа кода не удалось загрузить целевой файл, указанный для анализа. Это происходит, если файл не найден или доступ к файлу запрещен. Как правило, это происходит, если пользователь выбрал переопределение свойства входной сборки в Visual Studio или вызывает fxcopcmd.exe вручную.
