# <a name="list-of-available-services"></a>Список доступных служб

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] пакет SDK для Visual Studio поддерживает следующие службы. Некоторые пакеты предлагают собственные службы, которые не перечислены здесь. Например, языковые службы не имеют одного идентификатора GUID службы. Для поиска идентификатора GUID языковой службы в реестре необходимо использовать имя языка.

Используйте указанные здесь идентификаторы GUID службы или полученные из другого источника (например, языковых служб), чтобы получить основной интерфейс или интерфейсы, отображаемые для каждой службы.
